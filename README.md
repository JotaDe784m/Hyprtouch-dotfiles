# HyprTouch

> A modern, touch-first Hyprland configuration designed for Microsoft Surface devices.

HyprTouch is an open-source project focused on providing a polished, modular and easy-to-maintain Hyprland experience, with special attention to touchscreen devices.

## Vision

HyprTouch is more than a collection of dotfiles. The goal is to build a complete desktop experience consisting of:

- Modular Lua-based Hyprland configuration
- Quickshell interface
- hyprtouchd system daemon
- Easy installation and updates
- Touch-first workflow
- Clean and maintainable architecture
